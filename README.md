## Lokalde Calistirma

**Bu projede NPM ve ParcelJS kullanilmistir. Projeyi lokalde calistirmak icin cihazinizda npm ve nodejs yuklu olmalidir.**

### NPM Paketlerini yukleme

```js
npm install
```

### Parcel ile lokal server calistirma

```js
npm start
```
